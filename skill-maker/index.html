📖 SKILL COMPASS 사용 가이드

7단계만 따라 하면 나만의 Claude 스킬이 완성됩니다.

01

📝 주제 입력

회원가입이나 API 키 없이 스킬 주제와 핵심 작업만 입력합니다. 처음 쓰는 분도 예시 칩을 눌러 바로 시작할 수 있습니다.

02

🎯 세부 정보 보강

카테고리, 출력 형식, 페르소나, 대상 사용자를 선택하면 더 실무적인 스킬 초안이 만들어집니다.

03

⚡ 즉시 생성

[무료로 SKILL.md 초안 만들기] 버튼을 누르면 브라우저 안에서 바로 Claude용 스킬 초안이 생성됩니다.

04

⬇ 다운로드

결과 화면에서 .md 단일 파일 또는 README·참고자료가 포함된 .skill 패키지(ZIP)로 다운로드합니다. 보관함에 저장해두면 언제든 다시 불러올 수 있습니다.

05

🤖 Claude에 등록

방법 A — Claude 앱 > 설정 > 기능 > 스킬(Skills)에서 .skill 파일 업로드.
방법 B — SKILL.md 내용을 복사해 Claude Project의 지침(Instructions)에 붙여넣기.

06

🧪 트리거 테스트

스킬에 정의된 트리거 문장(예: "보도자료 써줘")을 Claude에 입력해 스킬이 자동 실행되는지 확인합니다. 잘 안 되면 트리거 힌트를 보강해 다시 생성하세요.

💡 07 — 반복 활용: 보관함에서 언제든 스킬을 불러와 수정·재다운로드할 수 있습니다. 팀원과 .skill 파일을 공유하면 조직 전체가 같은 품질의 결과물을 얻습니다.

❓ 자주 묻는 질문 (FAQ)

API 키 없이도 정말 되나요?
네. 이 공개용 버전은 Claude API를 직접 호출하지 않고, 입력한 내용을 바탕으로 브라우저 안에서 SKILL.md 초안을 즉시 만들어 줍니다. 이후 Claude에 붙여넣어 다듬으면 됩니다.
생성된 스킬은 어디에 쓰나요?
생성된 SKILL.md 내용을 Claude Project 지침에 붙여넣거나, .skill 패키지로 다운로드해 Claude의 Skills 기능에 업로드하면 됩니다.
더 고급스럽게 만들고 싶으면?
초안을 Claude에게 붙여넣고 “이 SKILL.md를 더 전문적으로 보강해줘”라고 요청하면 됩니다. 핵심 작업과 트리거 문장을 구체적으로 쓸수록 품질이 좋아집니다.
보관함 데이터는 어디에 저장되나요?
브라우저의 localStorage에 저장됩니다. 서버 전송이 전혀 없어 안전하지만, 같은 브라우저·같은 기기에서만 보입니다. 브라우저 데이터를 삭제하면 함께 지워지니 중요한 스킬은 [전체 내보내기]로 백업하세요.
비용이 발생하나요?
이 페이지에서 초안을 생성하는 것 자체는 무료입니다. 단, 생성된 스킬을 Claude에서 실제로 사용할 때는 사용자의 Claude 요금제 또는 API 사용 환경에 따라 비용이 발생할 수 있습니다.

📘 SKILL이란?

SKILL은 "Claude에게 업무 수행 방법을 가르치는 지침 파일"입니다.
한 번 만들어두면, 매번 긴 프롬프트를 쓰지 않아도 Claude가 정해진 절차·형식·품질 기준대로 일합니다.

😩 Before — 스킬이 없을 때

"보도자료 써줘. 제목은 이렇게, 리드문은 이렇게, 인용은 2개, 어조는 공식적으로, 분량은 A4 1장, 마지막에 회사 소개 넣고..." 를 매번 반복 입력. 사람·날짜마다 결과 품질이 들쭉날쭉.

😎 After — 스킬이 있을 때

"신제품 보도자료 써줘" 한 마디면 끝. 스킬에 내장된 페르소나·워크플로우·품질 체크리스트대로 항상 동일한 전문가급 결과물이 나옵니다.

🚀 만든 스킬 활용 가이드

API 키 없이 만든 초안을 Claude에서 실제 업무용 스킬로 활용하는 방법입니다.

01

초안 생성

스킬 주제와 핵심 작업을 입력하고 SKILL.md 초안을 만듭니다. 처음에는 완벽함보다 빠른 초안이 중요합니다.

02

Claude에서 다듬기

생성된 내용을 Claude에 붙여넣고 “이 스킬을 더 전문적으로 보강해줘”라고 요청하면 품질을 높일 수 있습니다.

03

Project 지침에 붙여넣기

Claude Project의 지침(Instructions)에 SKILL.md 내용을 붙여넣으면 해당 프로젝트 안에서 스킬처럼 사용할 수 있습니다.

04

.skill 패키지 저장

다운로드한 .skill 파일은 백업하거나 팀원과 공유할 수 있습니다. 조직 내 반복 업무 표준화에 유용합니다.

05

트리거 테스트

스킬에 적힌 예시 문장을 Claude에 입력해 원하는 방식으로 실행되는지 확인하고, 부족하면 트리거 문장을 추가하세요.

⚠ 안내
· 이 페이지는 공개용 초안 생성기입니다. 사용자의 API 키나 비밀번호를 요구하지 않습니다.
· 실제 Claude Skills 업로드 기능은 Claude 계정/요금제에 따라 제공 범위가 다를 수 있습니다.
· 중요한 업무에 쓰기 전에는 생성된 스킬 내용을 반드시 한 번 검토하세요.
